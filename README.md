# cssview

An app that wraps ShowcaseFX. ShowcaseFX is a view that show a preview of your JavaFX css file applied to all JavafX controls.

## Usage

```bash
git clone git@github.com:hohonuuli/cssview.git
cd cssview
mvn clean javafx:run
```
